# OldFMchess-js

## Run

Install Node.js first.

Then install http-server:

```bash
npm install -g http-server
```

Run:

```bash
npx http-server
```

Open:

```txt
http://127.0.0.1:8080
```

Or:

```txt
http://192.168.100.35:8080
```
